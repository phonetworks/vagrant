# vagrant

Preparation scripts for for [xenial-pho](https://app.vagrantup.com/phonetworks/boxes/xenial-pho) Vagrant box.

## Install

Download and install [Vagrant](https://vagrantup.com). Then:

```
vagrant init phonetworks/xenial-pho
vagrant up
```
